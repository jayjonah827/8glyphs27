## CLAIM TYPE DEFINITIONS (Research family only)

**Descriptive** — states what the data shows. No null model required.
Example: "The 39% value appears across two datasets 200 years apart."

**Inferential** — states that the observed pattern rejects a specified null model.
Example: "Clustering rejects uniform null at p < 0.001 (KS test, Levene test)."
Requires: null model stated, test identified, effect size reported.

**Theoretical / Mechanistic** — states a causal or structural explanation.
Example: "0.39 functions as a Hamiltonian ground state or structural attractor."
Requires: explicit labeling as theoretical until independently validated.

Rule: these three claim types must never appear in the same sentence or be used interchangeably in public-facing research language.
